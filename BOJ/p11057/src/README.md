## 문제 설명

<p>오르막 수는 수의 자리가 오름차순을&nbsp;이루는 수를 말한다. 이때, 인접한 수가 같아도 오름차순으로 친다.</p>
<p>예를 들어, 2234와 3678, 11119는 오르막 수이지만, 2232, 3676, 91111은 오르막 수가 아니다.</p>
<p>수의 길이 N이 주어졌을 때, 오르막 수의 개수를 구하는 프로그램을 작성하시오. 수는 0으로 시작할 수 있다.</p>
